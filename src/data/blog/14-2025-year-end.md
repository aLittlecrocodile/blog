不知道从什么时候开始，养成了年底写总结的习惯。也许是因为不总结就感觉这一年白过了，也许只是给自己一个交代。

今年是折腾得比较多的一年。

## 技术上

年初定了个目标：把 Go 真正搞透。之前写 Go 更多是在用，照着文档能跑起来就行，但对底层几乎没有感觉。这一年花了不少时间去啃 GMP 调度模型、channel 的底层实现、GC 的三色标记，写了一堆笔记，也交了一些智商税——有些"理解"到后来发现是错的，改了好几遍。

上半年开始真正接触 Kubernetes，最开始只是把它当成"一个更复杂的 Docker Compose"，直到真的踩了几次坑，才慢慢摸出点感觉。Prometheus + Grafana 那套监控体系也是这一年配起来的，PromQL 写着写着竟然有点上瘾。

下半年 AI 的浪潮实在太猛，绕不开。花时间研究了一阵 LLM 的工程化落地，做了一个多 Agent 协作的小项目，把监控告警这个场景和 LLM 结合了一下。说不上有多深，但算是有了自己的判断：AI 不会替代后端，但会把工具链改造得面目全非。

今年写博客的频率比预期低。很多东西脑子里想清楚了，真要落到文字上又发现漏洞一堆，只能继续去查。但这个过程本身是值得的——写文章逼着自己真的搞懂，而不只是"感觉懂了"。

## 生活上

今年健身的频率比去年高了很多，大概维持在每周三到四次。

健身这件事对我来说不只是身体层面，更像是一个情绪阀门——写代码写到钻牛角尖的时候，去撸一组铁回来，往往思路就通了。秋天备战秋招压力大的时候一度想停掉，后来发现停了反而更焦虑，就坚持下来了。

有一段时间作息很乱，三点睡十点起，连续两周。后来发现那段时间写的代码回头看一半要重构，效率根本没有自己以为的高。现在尽量维持十二点前睡，状态好了很多。

## 关于 2026

春招是眼下最具体的事。目标方向还是 Go 后端 / 云原生，希望能找到一个有真实技术挑战的团队，而不只是写增删改查。

不给自己立太多 flag 了，去年立了一堆，完成率感人。

就这样，2025，再见。
